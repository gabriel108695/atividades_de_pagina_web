O que eu aprendi sobre HTML (Capítulo 2)

O HTML é a base de tudo no frontend. Ele serve para montar a estrutura das páginas, tipo como se fosse o esqueleto de um prédio.

História básica

Descobri que o HTML começou lá em 1992 com o Tim Berners-Lee. Hoje em dia, a gente usa o HTML5 (que saiu em 2014) e o chamado "Living Standard", que é uma versão que está sempre evoluindo.

Como as tags funcionam

A gente usa elementos e atributos. Basicamente:

  • As tags abrem e fecham, tipo <h1> e </h1>.
  • O que fica no <head> o navegador não mostra na tela, como o título da aba.
  • Tudo o que a gente vê de verdade fica dentro do <body>.
  • Dá para colocar informações extras usando atributos, como o href para links.

Formatando textos e listas

Aprendi a fazer títulos do 1 ao 6. O 1 é o maior de todos. Também tem como deixar o texto em negrito com <b> ou itálico com <i>.

Sobre as listas, existem três tipos principais:

  1. Ordenadas: Aquelas que têm números (tag <ol>).
  2. Não ordenadas: Aquelas com bolinhas (tag <ul>).
  3. De definição: Para termos e explicações (tags <dl>, <dt> e <dd>).

Dica importante

O livro diz que para ver o código de qualquer site, é só usar as "Ferramentas do Desenvolvedor" no Chrome (no menu View > Developer).
