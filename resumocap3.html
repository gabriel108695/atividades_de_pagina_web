O que e css?

CSS é a linguagem responsável pela aparência visual de uma página web. Ele permite personalizar elementos HTML, como textos, listas, formulários e tabelas, deixando o site mais organizado e visualmente agradável.


3.1 Introdução

mostra como as folhas style são estruturadas em css e como podemos incluir no código html.


3.1.1 O Princípio do CSS.

Com CSS, é possível mudar cores, fontes, tamanhos, bordas, espaçamentos e vários outros aspectos visuais da página. Cada propriedade aceita apenas determinados valores válidos definidos pela própria linguagem CSS.


3.1.2 Incluindo css em html.

Existem três maneiras principais de usar CSS em uma página HTML:

  1. CSS Externo (External CSS)

    2. O código CSS fica em um arquivo separado com extensão .css. Depois, esse arquivo é conectado ao HTML. Essa é a forma mais organizada e mais usada em projetos.

  2. CSS Interno (Internal CSS)

    3. O CSS é escrito dentro do próprio arquivo HTML, na seção style dentro do head. É útil para estilos específicos de uma única página.

  3. CSS Inline (Inline CSS)

    4. O estilo é aplicado diretamente em um elemento HTML usando o atributo style. Geralmente é usado para pequenas alterações rápidas.
