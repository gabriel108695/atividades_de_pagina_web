O que e css?

CSS é a linguagem responsável pela aparência visual de uma página web. Ele permite personalizar elementos HTML, como textos, listas, formulários e tabelas, deixando o site mais organizado e visualmente agradável.

mostra como as folhas style são estruturadas em css e como podemos incluir no código html.


Princípio do CSS.

Com CSS, é possível mudar cores, fontes, tamanhos, bordas, espaçamentos e vários outros aspectos visuais da página. Cada propriedade aceita apenas determinados valores válidos definidos pela própria linguagem CSS.


Incluindo css em html.

Existem três maneiras principais de usar CSS em uma página HTML:

  1. CSS Externo (External CSS)

    2. O código CSS fica em um arquivo separado com extensão .css. Depois, esse arquivo é conectado ao HTML. Essa é a forma mais organizada e mais usada em projetos.

  2. CSS Interno (Internal CSS)

    3. O CSS é escrito dentro do próprio arquivo HTML, na seção style dentro do head. É útil para estilos específicos de uma única página.

  3. CSS Inline (Inline CSS)

    4. O estilo é aplicado diretamente em um elemento HTML usando o atributo style. Geralmente é usado para pequenas alterações rápidas.


Incluindo Arquivos CSS Externos (CSS Externo)

O uso de arquivos CSS externos é, na maioria das vezes, a melhor escolha para organizar o visual de um site. Isso acontece porque o CSS fica separado do HTML, deixando o código mais limpo e fácil de entender. Além disso, o mesmo arquivo CSS pode ser reutilizado em várias páginas do projeto, garantindo que todas tenham a mesma aparência e estilo visual. Outra grande vantagem é a praticidade na manutenção: quando for necessário mudar cores, fontes ou qualquer detalhe do design, basta alterar o arquivo CSS centralizado uma única vez. Assim, todas as páginas conectadas a ele serão atualizadas automaticamente, economizando tempo e mantendo a consistência do site.


Cascata, Especificidade e Herança

  • Cascata (Cascading): Determina a prioridade de aplicação quando múltiplas regras afetam o mesmo elemento. Se os seletores forem idênticos, a última regra definida no código substitui a anterior.
  • Especificidade: Se os seletores forem diferentes, o seletor mais específico ganha prioridade (por exemplo, um seletor de ID é mais forte que um de classe, que por sua vez é mais forte que um seletor de tag geral).
  • Herança: Algumas propriedades (como font-family e color) são herdadas pelos elementos filhos a partir dos elementos pai, enquanto outras (como border) não são herdadas.

Cores e Formatação de Texto

  • Definição de Cores: Podem ser aplicadas ao texto (color) ou ao fundo (background-color) utilizando nomes de cores, valores RGB, RGBA (com opacidade), valores hexadecimais ou HSL/HSLA.
  • Tipografia e Tipos de Letra: A propriedade font-family define o tipo de letra, permitindo listar opções secundárias (fallbacks) caso a principal não esteja instalada no computador. O tamanho é ajustado via font-size (usando píxeis, percentagens ou a unidade
  • Estilo e Alinhamento: O texto pode ser formatado em itálico (font-style), negrito (font-weight), maiúsculas/minúsculas (text-transform), sublinhado (text-decoration) e alinhado horizontalmente (text-align).
  • Estilização de Links: Utilizam-se pseudo-classes (:link, :visited, :hover, :active, :focus) para modificar o visual dos links dependendo do estado de interação do utilizador.

    • Sistemas de Layout (Disposição de Elementos)

    • Float Layout: Técnica mais antiga que usa a propriedade float (para alinhar elementos à esquerda ou à direita).
    • Flexbox Layout: Introduzido no CSS3, é um sistema unidimensional (funciona numa direção de cada vez, horizontal ou vertical). É altamente dinâmico, ajustando automaticamente o tamanho dos elementos filhos (flex items) dentro de um contentor (flex container) para preencher o espaço de forma flexível.
    • Grid Layout: Um sistema bidimensional mais recente e adequado para layouts complexos, permitindo alinhar elementos em linhas e colunas em simultâneo dentro de um contentor de grelha.

    Em resumo, e como um guia estruturado para quem deseja aprender a controlar o aspeto visual e a disposição espacial de uma página web moderna utilizando CSS.
